# Lexo

**Lexo** traduit le texte sélectionné dans n'importe quelle application macOS,
d'un simple raccourci clavier — **100 % en local**, sans envoyer vos données sur Internet.

## ⬇️ Télécharger

➡️ **[Télécharger la dernière version (.dmg)](https://github.com/Titi257/lexo/releases/latest/download/Lexo.dmg)**

- macOS 15 (Sequoia) ou plus récent
- Universel : Intel & Apple Silicon
- Gratuit

## Installation

1. Ouvrez le fichier `.dmg` téléchargé.
2. Glissez **Lexo** dans le dossier **Applications**.
3. Lancez Lexo et accordez l'autorisation **Accessibilité** au premier lancement
   (nécessaire pour lire la sélection).

## Utilisation

Sélectionnez du texte dans n'importe quelle app et appuyez sur **⌃⌥T**.
La traduction s'affiche dans une bulle près du curseur.

## Confidentialité

Lexo utilise le moteur de traduction d'Apple (Translation.framework) **directement
sur votre Mac**. Aucun texte n'est envoyé sur Internet, aucun compte, aucune clé API.
